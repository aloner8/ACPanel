# Frontend Modules

## `apps/web`

Angular admin console for ACPanel.

### Suggested feature modules

- `auth`
- `dashboard`
- `customers`
- `domains`
- `packages`
- `deployments`
- `scripts`
- `logs`
- `settings`

### Suggested shared UI areas

- API service layer
- auth interceptor
- layout shell
- notification system
- table and filter components
- form controls for domain and deployment config
